Compilers

I have, for the last year or so, been working on a compiler for a programming language of my own design. The programming language is called Grime and takes influence from Kotlin, Swift, C and Rust. The language supports both functional and OO styles of programming however all Objects are immutable. The use of immutable objects makes multithreaded programming.
